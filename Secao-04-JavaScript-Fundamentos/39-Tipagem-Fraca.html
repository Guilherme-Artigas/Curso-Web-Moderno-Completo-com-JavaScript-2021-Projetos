Tipagem Fraca

Variando Variáveis

JavaScript é um tipo de linguagem que podemos declarar uma variável e essa variável receber um valor inteiro, nas próximas linhas receber um valor real e nas próximas linhas receber um valor string, chamamos isso de tipagem fraca, a linguagem tem um dinamismo para se trabalhar com os tipos de dados muito grande, isso tem suas vantagens e desvantagens.

Nessa aula falamos sobre a tipagem do JavaScript, o Professor Leonardo comenta que existe o lado bom e o lado ruim da linguagem seguir esse caminho, ou seja, por um lado é uma grande vantagem, pois da liberdade a linguagem trabalhar de uma forma muito mais dinâmica dentro das aplicações, por outro lado fica muito mais difícil identificar bugs que possam eventualmente acontecer por conta dessa tipagem.
