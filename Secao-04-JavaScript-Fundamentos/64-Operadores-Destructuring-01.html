Operadores: Destructuring #01

Atribuição via desestruturação

A sintaxe de atribuição via desestruturação é uma expressão JavaScript que possibilita extrair dados de arrays ou objetos em variáveis distintas.

O operador destructuring tem duas formas de escrita quando trabalhamos com objeto usamos em sua sintaxe as chaves e quando trabalhamos com array usamos em sua sintaxe os colchetes.

Devemos ter cuidado para desestruturar um dado que esta aninhado, precisamos ter certeza de que esse dado existe, caso contrário ira retornar erro.
