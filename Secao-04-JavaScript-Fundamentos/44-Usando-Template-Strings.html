Usando Template Strings

Strings Aula 2

Nessa aula falamos sobre mais uma forma de se declarar uma frase dentro de um código, seguindo a mesma regra iniciando com crase e terminando com crase. É uma forma mais simples, elegante, deixa o código mais bonito, em situações que precisarmos de muitas concatenações o uso de template strings se aplica muito bem.

O uso dessa forma alternativa, mais simples e elegante, tambem permite coisas que declarações com aspas simples e duplas não permitem, um exemplo e a quebra de linhas no meio da declaração.
