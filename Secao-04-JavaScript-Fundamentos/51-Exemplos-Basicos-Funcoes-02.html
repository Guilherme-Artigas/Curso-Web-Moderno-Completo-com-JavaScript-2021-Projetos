Exemplos Básicos de Funções #02

Funções aula 02

Nessa aula o professor Leonardo comentou sobre atribuir uma função dentro de uma variável e poder que isso tem dentro de JavaScript, nem todas as linguagens têm essa possibilidade.

Armazenando uma função em uma variável

const imprimirSoma = function (p1, p2) {console.log(p1 + p2);};

Armazenando uma função arrow em uma variável

const soma = (p1, p2) => {return p1 + p2;};

Retorno implícito

const subtracao = (p1, p2) => p1 – p2;

Automaticamente quando não temos abertura e fechamento de chaves significa que teremos apenas uma sentença de código.
