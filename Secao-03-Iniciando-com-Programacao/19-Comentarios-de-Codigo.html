Comentários dentro do JS

Nessa aula aprendemos sobre a importância dos comentários dentro do código, pois, eles nos auxiliam tanto para o desenvolvimento, quanto para explicações posteriores, deixando assim de forma mais explicita para outros programadores qual é a ideia de execução daquele bloco de código ou a logica por trás de uma simples linha de comando, ajudando assim na manutenção do script.

Podemos definir comentários dentro do JavaScript de uma linha só, ou de um bloco contendo várias linhas.

// Comentário de uma única linha de comando.

/* Comentário de um bloco de linhas. */
