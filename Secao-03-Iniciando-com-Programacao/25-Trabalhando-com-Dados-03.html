Trabalhando com Dados Aula 03

Tratamento de Dados

Nessa aula vemos sobre especificamente sobre cada tipo de dado, como a linguagem JavaScript reconhece e trabalha com cada um deles.

Tipo de dado numérico

Dentro da linguagem JavaScript um valor numérico inteiro por exemplo: 6 e um número com ponto flutuante por exemplo: 5.2 é tratado da mesma forma, ou considerado dentro da mesma família, que é a Number ou números, em outras linguagens vemos diferenças entre esses tipos de números, onde a linguagem trata de forma específica cada um deles, números inteiros/int e números com pontos flutuantes/float.

Uma observação interessantíssima dessa aula, é que o professor Leonardo ensinou a funcionalidade interna do JavaScript chamada typeof passando o valor literal ou a variável que queremos consultar, descobrimos o seu tipo, porém quando declaramos uma variável e atribuímos um valor a ela e paramos com o cursor encima dessa variável temos o mesmo resultado.

Tipo de dado verdadeiro ou falso / Boolean (True / False)

Esse tipo de dado está muito ligado a estruturas condicionais e laços de repetições dentro das linguagens de programação, por que a resposta pode ser verdadeiro ou falso, porém também podemos declarar variai veis com o valor literal explícito e depois podemos alterá-lo no decorrer do script ou da aplicação, tudo vai depender única e exclusivamente da lógica desenvolvida para aquele código.

Tipo de dado String

Esse tipo de dado é usado dentro das linguagens para armazenar textos, sendo eles simples letras ou palavras ou frases, tudo ira depender da nossa necessidade e da lógica para trabalharmos com o desenvolvimento do código, também usamos esse tipo de dado para saídas de informações na tela, muito útil para pedir ao usuário para digitar alguma informação que precisamos, por exemplo.
