Conhecendo o Queridão VScode

Conhecendo o nosso editor de códigos.

Nessa aula foi ensinado a configurar a extensão Code Runner para que sempre que o arquivo .js seja executado ele salve o arquivo antes da execução e sempre mostre o resultado atual limpando a tela da execução anterior.

Ctrl + Alt + N = executa o código

Na engrenagem da extensão Code Runner:

Extension Settings > Clear Previuos Output

(Limpa a execução anterior e mostra sempre a saída atual.)

Save File Before Run

(Salva o arquivo antes da execução)
