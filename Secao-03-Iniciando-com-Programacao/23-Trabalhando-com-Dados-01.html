Trabalhando com Dados Aula 1

Aula falando sobre tipos de dados

Dados são importantes dentro de todas as linguagens de programação, pois são eles que são usados para trabalharmos com os resultados, através da coleta de informações vindas do usuário da nossa aplicação, nós programadores escrevemos scripts que transformam esses dados coletados em outros valores, para muitas diferentes finalidades.

Dica da aula: Quando colocamos valores diretamente explícitos nas sentenças de código, dizemos que é um valor literal.

Nessa aula vemos sobre a declaração de variáveis, as variáveis dentro das linguagens de programação são os recursos que os programadores têm para fazer o tratamento dos dados que são preenchidos por usuários ou no decorrer dos scripts da aplicação.

Podemos somente declarar a variável dando um nome a ela, ou podemos declará-la e também atribuir valor.

Exemplo de declaração de variável, sem atribuição de valor: var exemplo;

Exemplo de declaração de variável com atribuição de valor: var exemplo = 5;

Veja que no segundo exemplo colocamos o sinal de igual, na maioria das linguagens de programação o sinal de igual é um sinal de atribuição de valor a denominada variável, no segundo exemplo lemos que a variável exemplo esta recebendo o valor numérico inteiro 5.
