Tipos em JavaScript: Boolean

Verdadeiro ou Falso (True / False)

Nessa aula o assunto é sobre valores booleanos e a importância desse tipo de valor dentro da programação, é dentro do JavaScript temos os valores true ou false como literais, 0 e 1, e operações que retornam esses valores de forma implícita como as estruturas condicionais if, elseif, switch, nesses casos é sobre tomadas de decisões, por exemplo caso algo tenha resultado como verdadeiro será executado determinado bloco de código, caso seja falso, será executado outro bloco de código e isso nos abre uma árvore muito grande de possibilidades dentro das nossas aplicações. Estruturas de laços ou estruturas de repetição, nesses casos para que o laço continue sendo executado ou para que o looping seja interrompido quando determinada condição for considerada como false.

Valores verdadeiros, porém, escondidos.

Todos os números inteiros com exceção do 0 são considerados como verdadeiro dentro do JavaScript.

Uma String preenchida com, pelo menos, um valor de caractere, também é considerada como true em JavaScript.

Uma coleção de valores (array), mesmo que vazio, tambem é considerado como true dentro da nossa querida linguagem JS.

Um objeto, tambem é considerado como true.

Infinity, tambem é considerado como true.

Valores Falsos, porém, escondidos.

O valor 0 é considerado como false.

Uma String vazia é considerada como false.

O valor null e undefined também é considerado como false.

O professor Leonardo mostrou na aula, que podemos fazer diretamente operações lógicas dentro de um console.log(); vamos ao exemplo.

Console.log(!!(‘’ || null || 0 || ‘ ’));

(O resultado dessa operação será o primeiro valor verdadeiro/true que aparecer.)
