Tipos em JavaScript: String

String

Nessa aula falamos sobre String dentro da programação e mais específico dentro da linguagem JavaScript, é um tipo muito importante para a programação em si, pois, assim como temos números com o tipo Number, String são tipos que trazem a formação de frases e armazenamento de letras e palavras, algo que é muito importante para tratamento de dados como um todo, o professor Leonardo comentou que nem só de números vive uma aplicação e que no JavaScript podemos começar uma cadeia de caracteres e dizer que ela é uma frase começando e terminando a frase com aspas simples ou começando e terminando a frase com aspas duplas e a partir da versão mais nova do EcmaScript podemos delimitar também com o símbolo da crase, seguindo a mesma regra, começando e terminando com a crase. O uso da crase é para quando precisarmos fazer uso de template String, o professor comentou que teremos uma aula especifica sobre isso mais adiante no curso.

Vimos nessa aula algumas funções internas do JS relacionadas ao tipo String.

Const escola = 'Cod3r';

console.log(escola.charAt(4));

(Retorna a letra que está no índice 4 de determinada variável/constante. Nesse exemplo vai retornar r.)

console.log(escola.charCodeAt(3));

(Valor contido dentro do índice, será retornado de acordo com a tabela ASCII. Nesse exemplo vai retornar 51.)

console.log(escola.indexOf('3'));

(Ira exibir o índice em que esta contido o valor mencionado, caso não encontre o valor retornado será -1. Nesse exemplo 3.)

console.log(escola.substring(1));

(Retorna a frase a partir do valor passado como índice. Nesse caso 'od3r'.)

console.log(escola.substring(0, 3));

(Podemos também passar dois valores como parâmetro, dizendo o início e o final (não ira incluir o índice 3). Nesse caso ira retornar Cod.)

console.log('Escola'.concat(escola).concat('!'));

(Diferente de Number podemos chamar a função direta, nesse caso teremos valores concatenados formando uma frase 'Escola Cod3r!'.)

Lembrando: valor literal é qualquer valor que foi mencionado, porém, ele não esta contido dentro de uma variável ou constante, podendo ele ser número, letras ou um conjunto de palavras.

console.log(escola.replace(3, 'e'));

(Substituindo determinado caractere passando como parâmetro o índice em que o valor se encontra e qual carcere queremos colocar no lugar.)

console.log('Ana,Maria,Pedro'.split(','));

(Transforma a frase em um array, passando o valor de parâmetro como separador.)
