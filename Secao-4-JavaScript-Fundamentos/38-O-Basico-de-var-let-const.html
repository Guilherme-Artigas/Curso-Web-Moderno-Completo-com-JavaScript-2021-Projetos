O Básico sobre var, let e const

Falando sobre armazenamento de dados

Nessa aula vemos sobre a declaração de variáveis usando 3 possíveis declarações dentro do JavaScript, existem diferenças entre cada uma delas, porém não vamos abordar essas diferenças nessa aula.

Declarando variáveis com var

var exemplo_com_var;

Declarando variáveis com let

let exemplo_com_let;

Declarando "variáveis" com const

const exemplo_com_const;
