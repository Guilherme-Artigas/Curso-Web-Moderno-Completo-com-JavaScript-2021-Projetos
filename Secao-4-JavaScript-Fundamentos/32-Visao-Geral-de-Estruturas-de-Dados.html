Estruturas de Dados

Nessa aula falamos sobre estrutura de dados, o professor Leonardo explica que o número 3, a letra P, o número 3.14, o valor booleane True ou False, todos eles são dados, mais existe diferença entre estrutura de dados e os dados em si.

Levantamos a pergunta na aula, qual é o objetivo da estrutura de dados? A resposta é que estrutura de dados esta dividido em 2 tópicos organizar e administrar, por exemplo na frase “saiu a lista dos aprovados” nessa frase podemos perceber a palavra lista, ou seja, organizar os nomes em uma lista, outro exemplo seria a lista de supermercado.

O professor comentou que lista, fila são estruturas de dados bem tradicionais nas linguagens de programação

Na aula também falamos sobre FIFO e LIFO:

FIFO do inglês é First in First out, o primeiro a entrar é o primeiro a sair.

LIFO do inglês é Last in First out, o último a entrar é o primeiro a sair.

Estrutura de árvore

Um bom exemplo desse tipo de estrutura é a forma que o Windows organiza os arquivos do sistema operacional, é como se fosse uma escada invertida.
