Tipos em JavaScript: Array

Array

Nessa aula aprendemos sobre a importância da estrutura array dentro da linguagem, ela é uma estrutura unidirecional que conseguimos agrupar vários dados dentro de uma única estrutura, pode ser números, pode ser letras ou palavras, nomes, idade de muitas pessoas, enfim… serve para auxiliar programadores a desenvolver as aplicações de forma mais eficiente. O acesso de cada dado contido dentro de array é feito por índice.

Como JavaScript é uma linguagem fracamente tipada, a estrutura de array aceita vários valores de vários tipos diferentes, número, String, Boolean entre outros, o professor comentou que apesar de conseguirmos fazer isso, não é considerada uma boa prática, o ideal é criar cada estrutura contendo um tipo de dado.

Exemplos

const valores = [7.7, 8.9, 6.3, 9.2];

console.log(valores[0], valores[3]);

(Retorno do console será: 7.7, 9.2)

console.log(valores[4]);

(Esse exemplo foi dado para reforçar que JavaScript não retorna erros, como em outras linguagens.)

valores[4] = 10; // Atribuindo um novo valor de forma literal.

console.log(valores);

(Retorno do console será: 7.7, 8.9, 6.3, 9.2, 10)

console.log(valores.length)

(Retorna no console, o tamanho do vetor, nesse caso: 5 posições/índices)

console.log(valores.push({id: 3}, false, null, ‘teste’));

(.push acrescenta valores ao final da lista, nesse exemplo, vemos que esses valores podem ser, objeto, booleano, null, String respectivamente.)

console.log(valores.pop());

(.pop retira do array o último item.)

Outra forma de se retirar valores de dentro de um array é com o comando delete valores[0].

(Nesse caso será retirado o valor que está na posição 0 do array.)

console.log(typeof valores);

(Comprovando que array dentro de JS é considerado do tipo object).
