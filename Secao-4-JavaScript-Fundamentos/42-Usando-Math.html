Usando Math

Objeto Math dentro do JavaScript

Nessa aula aprendemos alguns usando do objeto Math dentro da linguagem JavaScript. O professor Leonardo comentou que dento de JS não existe o operador de exponenciação, existe uma função interna que é o Math.pow() dentro dessa função passaremos dois argumentos e o retorno dela é o equivalente ao operador de potência de outras linguagens.
