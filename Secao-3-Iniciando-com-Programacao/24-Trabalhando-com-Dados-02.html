Trabalhando com Dados Aula 02

Tratamento de dados

Nessa aula, o professor Leonardo está enfatizando sobre a importância de se definir bons nomes as variáveis, por exemplo: evitar ao máximo abreviações, sempre que possível definir o nome completo com uma ou duas palavras, sem pontos, vírgulas e acentuações, no caso do JavaScript a linguagem aceita variáveis com identificadores ou nomes com acentuações, porém não é considerado uma boa prática.

O professor ensinou a concatenação de duas variáveis string com o sinal de +, o sinal de + pode ter ações diferentes dependendo das variáveis que estão adjacentes a ele, por exemplo, se for duas variáveis de tipo numérico o sinal de mais ira agir como soma de valores, se as duas variáveis forem string o sinal de mais ira juntar as palavras ou frases.
